CurseForge is one of the oldest Minecraft hosting platforms, housing thousands of "Bukkit" plugins and mods. The orchestrator uses the [CurseForge API](https://console.curseforge.com/) to resolve and download these files.

<Callout title="Important" type="warn">
CurseForge requires an **API Key**. Unlike other providers, you cannot use this provider without
setting the `CURSEFORGE_API_KEY` environment variable. You can obtain a free key by creating an
account at the [CurseForge Console](https://console.curseforge.com/).

Note that the API key may include dollar signs, which might need to be [escaped properly](https://docs.docker.com/reference/compose-file/interpolation)
in your YAML file.

</Callout>

## Usage

Use the `curseforge:` prefix followed by the project slug or ID.

```yaml
environment:
  CURSEFORGE_API_KEY: your_api_key_here
  PLUGINS: |
    curseforge:vault@latest
    curseforge:chunky-pregenerator@408295
```

<Callout title="Hint" type="info">
  You can use either the **Project Slug** (found in the URL) or the numeric **Project ID** (found in
  the sidebar). We recommend using the **Project Slug** for readability. ![CurseForge Project
  Slug](/curseforge_project_slug.png) OR (not recommended) ![CurseForge Project
  ID](/curseforge_project_id.png)
</Callout>

## Version Resolution

CurseForge supports standard version aliases, but **specific versions must be numeric File IDs**.

| Alias                      | Resolution                                                  |
| :------------------------- | :---------------------------------------------------------- |
| `@latest` or `@stable`     | Resolves to the most recent **Release** version.            |
| `@experimental` or `@beta` | Resolves to the most recent file (Release, Beta, or Alpha). |
| numeric ID (e.g. `408295`) | Resolves to the exact file matching the **File ID**.        |

<Callout title="Hint" type="info">
  You can find the specific **File ID** in the URL of a file's detail page or by hovering over the
  name in the **Files** tab. ![CurseForge File ID](/curseforge_file_id.png)
</Callout>

## The Force Flag (`!`)

The CurseForge provider defaults to assuming compatibility only with Bukkit-based servers (e.g. **PAPER** and **FOLIA**). If you use it with a different server type (like Velocity), the orchestrator will raise an error.

Use the `!` flag to bypass compatibility and platform checks:

```yaml
PLUGINS: |
  curseforge:12345@latest!
```
